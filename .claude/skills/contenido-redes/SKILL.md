# Skill: contenido-redes

Genera contenido para Instagram (y otras redes) para Guías Secretas.

## Cuándo usar esta skill
- Cuando necesites posts, reels, carruseles o stories para Instagram
- Cuando quieras un calendario de contenido semanal o mensual
- Cuando necesites adaptar contenido de las guías para redes

## Cómo invocarla
Dime:
1. **Tipo de contenido:** post estático, carrusel, reel (guión), story
2. **Tema o destino:** (ej: "Dolomitas", "Danubio", "cómo hacer una guía de viaje")
3. **Objetivo:** awareness, venta directa, captación de leads, engagement
4. **Cantidad:** cuántas piezas necesitas

Si no especificas, asumo: carrusel de 5-7 slides para Instagram, objetivo awareness, 1 pieza.

## Proceso

### Para un post / carrusel
1. Hook — primera slide que detiene el scroll (pregunta, dato, afirmación fuerte)
2. Desarrollo — 3-6 slides con el contenido de valor
3. CTA — última slide con llamada a la acción clara
4. Caption — texto del post con hook, desarrollo resumido y CTA
5. Hashtags — 5-10 relevantes (viajes + destino específico + nicho)

### Para un guión de reel
1. Hook (0-3s) — qué promete el vídeo
2. Desarrollo (3-45s) — puntos clave, ritmo rápido
3. CTA final (últimos 3s)

### Para un calendario semanal
- Lunes: contenido educativo / valor sobre destino
- Miércoles: detrás de cámaras o proceso de creación de la guía
- Viernes: venta o captación (promo, testimonio, beneficio directo)
- Opcional domingo: inspiración / wanderlust

## Tono
Directo, personal, con autoridad. Nada genérico. Quien lee siente que hay un experto detrás.
Sin emojis en el copy principal. Se pueden usar con moderación en hashtags si es necesario.

## Output esperado
Copy listo para copiar y publicar. Sin explicaciones adicionales salvo que las pidas.
