# Skill: paginas-seo

Crea páginas optimizadas para SEO que atraigan tráfico orgánico hacia Guías Secretas.

## Cuándo usar esta skill
- Crear una página de destino SEO para un destino (ej: "qué ver en las Dolomitas")
- Escribir un artículo de blog orientado a búsqueda
- Optimizar una página existente para palabras clave concretas
- Planificar una estrategia de contenido SEO para un destino

## Cómo invocarla
Dime:
1. **Destino o tema:** (ej: "Dolomitas", "viajar al Danubio en coche")
2. **Keyword objetivo:** si la tienes. Si no, la busco yo.
3. **Tipo de página:** artículo de blog, landing de destino, guía completa
4. **Intención de búsqueda:** informacional (aprendo), comercial (comparo), transaccional (compro)

## Proceso

### 1. Keyword research básico
- Keyword principal (alto volumen, intención clara)
- 3-5 keywords secundarias (long tail, preguntas relacionadas)
- Intención de búsqueda confirmada

### 2. Estructura de la página
```
H1: [Keyword principal + ángulo único]
Introducción (150-200 palabras): problema + promesa
H2: [Subtema 1]
H2: [Subtema 2]
H2: [Subtema 3]
...
CTA integrado: enlace o mención a la guía de pago
Conclusión con CTA final
```

### 3. Elementos SEO incluidos
- Title tag (máx 60 caracteres)
- Meta description (máx 155 caracteres)
- Sugerencia de URL slug
- Texto alt de imágenes (placeholders)
- Internal linking sugerido

### 4. CTA hacia el producto
Cada página SEO apunta a la guía de pago del destino o al curso Guía de Guías.
La transición del contenido gratuito al producto de pago tiene que ser natural, no forzada.

## Tono
Autoridad. El lector llega buscando información y encuentra a alguien que sabe más que cualquier blog genérico.
Directo, con opinión propia. Sin listas de "10 cosas que ver" sin criterio.

## Output esperado
Página lista para publicar: título, meta, URL, estructura completa con copy.
