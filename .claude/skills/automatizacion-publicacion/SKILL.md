# Skill: automatizacion-publicacion

Diseña flujos de automatización para publicar y distribuir contenido en web y redes sociales.

## Cuándo usar esta skill
- Diseñar un flujo de publicación semanal o mensual
- Automatizar la distribución de contenido entre plataformas
- Planificar un sistema de reutilización de contenido (repurposing)
- Integrar GHL con Instagram u otras plataformas

## Cómo invocarla
Dime:
1. **Qué quieres automatizar:** (ej: "publicar en Instagram 3 veces por semana", "distribuir blog a email")
2. **Herramientas disponibles:** GHL, Make/Zapier, Buffer, Meta Business Suite, etc.
3. **Frecuencia:** diaria, semanal, en lanzamientos
4. **Punto de partida del contenido:** ¿dónde se crea primero? (blog, GHL, Canva, etc.)

## Flujos estándar para Guías Secretas

### Flujo de contenido semanal (básico)
```
Lunes: Crear contenido en Canva (carrusel o imagen)
  → Programar en Meta Business Suite (miércoles, viernes, domingo)
  → Guardar en carpeta de referencias/examples/
```

### Flujo blog → email → redes
```
Publicar artículo SEO en web
  → GHL: trigger automático envía email con enlace al artículo
  → Extraer 3 puntos clave del artículo → post de Instagram
  → Story con link al artículo completo
```

### Flujo de lanzamiento (guía o curso)
```
Semana -2: contenido de calentamiento en Instagram (3 posts)
Semana -1: secuencia de emails de preventa (GHL)
Día 0: apertura → email de apertura + post en Instagram
Días 1-5: emails de nurturing + stories diarias
Día cierre: 2 emails (mañana y tarde) + post de urgencia
Post-lanzamiento: secuencia post-compra en GHL
```

## Lo que produce esta skill
- Diagrama del flujo (texto estructurado, no imagen)
- Pasos concretos con herramienta responsable de cada uno
- Triggers y condiciones de cada automatización
- Notas de implementación en GHL si aplica

## Limitación actual
Sin MCPs conectados, los flujos se documentan para implementación manual o mediante las herramientas nativas de GHL y Meta. Cuando se conecten integraciones, esta skill se actualiza.
