# Skill: embudo-ventas

Diseña y redacta embudos de venta completos para Guías Secretas. Se implementan en GHL.

## Cuándo usar esta skill
- Crear un embudo nuevo desde cero para una guía o el curso
- Revisar y mejorar un embudo existente
- Diseñar la arquitectura de páginas y flujo de un funnel

## Cómo invocarla
Dime:
1. **Producto:** guía específica o curso Guía de Guías
2. **Tipo de embudo:** lead magnet, venta directa, lanzamiento, evergreen
3. **Tráfico de entrada:** Instagram, SEO, email, ads
4. **¿Qué tienes ya hecho?** (si hay páginas o emails existentes, dímelo)

## Arquitectura estándar

### Embudo de guía (venta directa, €20-30)
```
Tráfico (Instagram / SEO)
  → Página de captación (lead magnet opcional)
  → Página de ventas
  → Checkout (GHL)
  → Thank you page + upsell (otra guía o curso)
  → Secuencia post-compra (3 emails)
```

### Embudo de curso (€29-57)
```
Tráfico
  → Opt-in (lead magnet: mini-guía gratuita o clase gratuita)
  → Secuencia de nurturing (5-7 emails)
  → Página de ventas del curso
  → Checkout (GHL)
  → Thank you page + order bump u OTO
  → Secuencia post-compra
```

## Lo que produce esta skill
Para cada fase del embudo:
- **Arquitectura:** qué páginas hay, en qué orden, qué pasa en cada transición
- **Copy de páginas:** headline, subheadline, bullets de beneficios, CTA, prueba social (placeholders)
- **Notas de implementación GHL:** qué tipo de página/flujo usar en cada paso

## Tono del copy
Directo, con autoridad, orientado a beneficios concretos.
El lector sabe exactamente qué obtiene y por qué vale el precio.
Sin hype vacío. Sin promesas exageradas.
